# About RustApps

A curated collection of lightweight applications and utility tools developed using the **Rust** programming language. This repository serves as a centralized hub for various high-performance, memory-safe, and standalone tools.
The applications in this project are cross-platform. Windows binaries are available for download. Users on other platforms (e.g. Linux, macOS) should compile from source code.
